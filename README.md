# 🎬 Movie Rating Prediction | CodSoft Internship

## 📌 Project Overview
Predicted IMDb ratings of Indian movies using features like genre, director, and cast.

## 📂 Dataset
- [IMDb Indian Movies Dataset](https://www.kaggle.com/datasets/adrianmcmahon/imdb-india-movies)

## 🛠️ Tools Used
- Python, pandas, scikit-learn, matplotlib

## ✅ Steps Done
- Data cleaning and preprocessing  
- Label encoding of categorical features  
- Linear Regression model  
- Visualization of actual vs predicted ratings
